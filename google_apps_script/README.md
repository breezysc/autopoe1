# Google Apps Script 数据转置工具

## 功能描述

该插件用于将Google Sheets中的纵向数据（列）转置为横向数据（行），并创建新的表格副本。

## 部署步骤

1. 打开 [Google Apps Script](https://script.google.com/)
2. 点击"新建项目"
3. 将 `Code.gs` 的内容复制到脚本编辑器中
4. 点击"文件" > "新建" > "HTML文件"
5. 命名为 "index"，并将 `index.html` 的内容复制进去
6. 点击"部署" > "新部署"
7. 选择"网络应用"
8. 设置以下参数：
   - 执行身份：以你自己的身份执行
   - 谁可以访问：根据需要选择（建议选择"任何人"）
9. 点击"部署"并授权
10. 复制生成的URL，这是你的应用访问地址

## 使用方法

1. 打开部署生成的URL
2. 填写以下信息：
   - 源 SpreadSheet ID：要读取数据的表格ID
   - 点击"获取Sheet列表"按钮，然后选择要读取的子表
   - 复制范围：纵向数据的范围（例如：Q3:Q19）
   - 目标起始坐标：数据写入的起始单元格（例如：B2）
   - 目标文件夹 ID：新表格要保存的文件夹ID
3. 点击"执行转置操作"按钮
4. 操作完成后，会显示成功消息和新表格的ID

## 核心功能

1. **数据转置**：将纵向数组（如 [[v1],[v2],[v3]]）转换为横向数组（如 [[v1, v2, v3]]）
2. **自动创建副本**：在目标文件夹下创建源表的副本
3. **横向写入**：从指定起始坐标开始横向写入转置后的数据
4. **自动命名**：新文件名使用源Sheet的名称，并且B2单元格也会写入该名称

## 注意事项

- 确保你有足够的权限访问源表格和目标文件夹
- 输入的范围格式必须正确（如 Q3:Q19）
- 目标文件夹ID必须有效
- 操作可能需要一些时间，取决于数据量大小
